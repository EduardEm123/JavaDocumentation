# JavaDocumentation
  Zoo Management System — система моделирования и управления животными в зоопарке. Это консольное Java-приложение, предназначенное для демонстрации принципов объектно-ориентированного программирования (наследование, инкапсуляция, полиморфизм) на примере управления зоопарком.Программа моделирует структуру зоопарка, позволяет добавлять животных, управлять их поведением и выводить информацию о каждом из них.

Основные возможности:
  1. Создание и управление экземплярами животных (например, львов);
  2. Отображение информации о животных;
  3. Управление зоопарком через основной класс ZooSystem.
Установка и запуск:
  1. Наличие Java Development Kit (JDK) версии 17 или выше, терминал или любая IDE (IntelliJ IDEA, Eclipse, VS Code с Java-плагином).
  2. Склонируйте репозиторий или скачайте файлы:
  3.Запустите основной класс ZooSystem. После запуска программа отобразит основное меню или результаты моделирования зоопарка.
Пример использования:
  Пример вывода программы:
    Добро пожаловать в Zoo Management System!
    Добавляем нового льва...
    Лев Симба: возраст 5 лет, вес 190 кг.
    Рычание: Roooar!
  Можно расширить добавив несколько новых классов животных.
Структура репозитория:
  ZooSystem.java - Главный класс для запуска кода
    ZooManager.java - Класс, отвечающий за управление коллекцией животных
    Animal.java - Абстрактный базовый класс Animal предназначен для описания общих свойств и поведения всех животных в системе.
    Lion.java - Класс, описывающий льва (наследует общий класс Animal)
    Dolphin.Java - Класс, описывающий дельфина (наследует общий класс Animal)
    Elephant.java - Класс, описывающий слона (наследует общий класс Animal)
