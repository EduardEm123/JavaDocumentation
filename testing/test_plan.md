1. Цель тестирования
   
            - Проверить корректность работы классов и модулей системы зоопарка:
            
            - Создание и взаимодействие объектов животных (Animal, Lion);
            
            - Управление коллекцией животных (ZooManager);
            
            - Выполнение действий животных (makeSound, performAction);
            
            - Корректный запуск приложения (ZooSystem).
2. Объекты тестирования

              Компонент:	  |  Описание:
              ____________________________________________________________
              Animal	      |  Абстрактный базовый класс для всех животных.
   
              Lion	          |  Подкласс Animal, реализующий поведение льва.
   
              ZooManager	  |  Управляет списком животных и их действиями.
   
              ZooSystem	  |  Главный класс для запуска программы.
4. Типы тестов
   
        Тип теста:                         |  Цель:
        ____________________________________________________________________________________
        Модульное тестирование             | Проверка отдельных методов классов
   
        Интеграционное тестирование        | Проверка взаимодействия между классами
   
        Функциональное тестирование        | Проверка соответствия программе требованиям
   
        Нагрузочное тестирование           | Проверка устойчивости при большом числе животных
