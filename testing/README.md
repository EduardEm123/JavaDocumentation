# Документация проекта Zoo Management System
Структура:

      Файл                  | Назначение                                                                                   
      ___________________________________________________________________________________________________________
        test_plan.md        | План тестирования. Содержит цели, сценарии и стратегию проверки работы программы.          
        test_report.md      | Отчёт о тестировании.    
        feedback_summary.md | Анализ отзывов пользователей. Сводка оценок, предложений и уровня удовлетворённости.          
        issues.md           | Список найденных ошибок и предложений по улучшению системы.

Назначение папки testing
      
    Папка testing содержит всю документацию проекта Zoo Management System, включая технические, тестовые и аналитические материалы.
    Эти документы используются для контроля качества, анализа отзывов пользователей и планирования улучшений системы.
